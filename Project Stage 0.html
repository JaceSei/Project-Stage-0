The basics of the web

The internet can be easily understood as a web of computers linked to one another. HTTP Requests are itiated by a user from their web browser in order to communicate with servers. A server locates the appropriate HTML document and then sends it back to the browser to be displayed on screen for the user.

HTML

The world wide web is basically a collection of html documents. HTML or Hypertext Markup Language is made up of Text content (what you see), Markup (what it looks like), Reference to other documents (e.g. images and videos) and Links to other documents (e.g. PDFs).

HTML Markup

Markup consists of using opening and closing tags with content in between to create specific elements in a HTML document. e.g. Open tag - content - closing tag. This can be used for something as simple as making text italic or creating the body of a HTML document.

HTML Attributes

Anchor tags or "A" tags are used for making links. They include a "value" in the opening tag e.g. a URL for linking to other HTML documents. Image tags or "IMG" tags are similar as they include a "value" in the opening tag, but image tags do not require a closing tag as they have no content. This type of tag is also referred to as a "Void Tag" which means no content.

Inline / Block Elements

HTML elements can be either Inline (will be displayed on one line) or Block (makes an invisible box around the content inside them).

Line Break = Inline

Paragraph = Block

Span = Inline

Div = Block
